# Pomodoro
A Mac app to suit my pomodoro needs. Released version 1.0.0. Will update in future.

### Details

* Xcode 3.2-compatible, but written using Xcode 7.3.1.
* Written in Swift 2.2, targeting x86_64-apple-macosx10.9.
